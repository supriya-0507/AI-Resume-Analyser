# AI Resume Analyzer

AI-powered Resume Analyzer built using FastAPI and Streamlit.

## Features

- Resume PDF Parsing
- ATS Score Calculation
- Skill Matching
- Missing Skills Detection
- Resume Score
- Recommendations
- Skill Match Charts
- Download Report

## Tech Stack

- Python
- FastAPI
- Streamlit
- Plotly
- Pandas
- PDFPlumber

## Run Backend

uvicorn backend.main:app --reload

## Run Frontend

streamlit run app.py
